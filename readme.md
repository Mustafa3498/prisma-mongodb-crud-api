# 🚀 Prisma + MongoDB CRUD API

This is a simple REST API built using **Node.js, Express, Prisma ORM, and MongoDB**.  
It demonstrates basic CRUD operations (Create, Read, Update, Delete).

---

## 📌 Tech Stack

- Node.js
- Express.js
- Prisma ORM
- MongoDB (Compass / Atlas)
- Postman (for testing)
